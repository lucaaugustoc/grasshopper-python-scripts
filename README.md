# Grasshopper Python Scripts 🐍🟩

Uma biblioteca pessoal de ferramentas e automações desenvolvidas em Python (`ghpython`) para o Grasshopper 3D e Rhino. 

O objetivo deste repositório é documentar a lógica de programação aplicada à arquitetura, unindo o design paramétrico com a eficiência na produção de projetos.

## 🗂️ Categorias de Scripts

Aqui estão os principais tipos de ferramentas que você encontrará neste repositório:

* **Automação de Tarefas:** Scripts para acelerar processos repetitivos (renomear camadas, organizar listas complexas, etc).
* **Geração de Geometria:** Ferramentas para criação de geometrias paramétricas complexas.
* **Análise e Dados:** Scripts para extração de áreas, quantitativos e integração de dados com outros softwares de arquitetura.
* **Análise e Estratégias Bioclimáticas:** Scripts focados em otimização ambiental e integração climática passiva nos projetos.
  * **Otimização de Proteções Solares (Brises):** Algoritmos para calcular a inclinação, profundidade e espaçamento ideais de brises baseados em vetores solares específicos do terreno.
  * **Tratamento de Dados Climáticos (EPW):** Scripts para extrair, filtrar e estruturar dados de radiação, temperatura e ventos de arquivos `.epw`, facilitando a integração com plugins de simulação como o Ladybug.
  * **Análise Geométrica de Insolação:** Ferramentas para calcular rapidamente a porcentagem de sombreamento em fachadas ou horas de sol direto ao longo do ano, auxiliando na tomada de decisão nas fases iniciais do projeto executivo.

## 🚀 Como usar

1. Faça o download do arquivo `.py` desejado.
2. No Grasshopper, adicione um componente `GhPython Script`.
3. Dê um duplo clique no componente, apague o código padrão e cole o código baixado.
4. Conecte os *Inputs* e *Outputs* conforme as instruções comentadas dentro de cada código.
